Portfolio GitHub Pages Ready

File struktur untuk GitHub Pages:

  • index.html - File utama website (rename dari index-github.html)
  • images/ - Folder berisi semua foto Anda
  • README.md - Dokumentasi repository

Langkah Upload ke GitHub:

  1. Download file index.html (yang sudah diperbaiki)
  2. Download folder images/ dengan semua foto
  3. Download README.md
  4. Upload semua file ke repository GitHub Anda
  5. Aktifkan GitHub Pages di Settings
